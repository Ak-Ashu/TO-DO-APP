# task01-AshishKumar
## AI TO-DO APP

A modern and interactive To-Do List web application built using Flask, HTML, CSS, and JavaScript with AI-assisted development. This project demonstrates core backend engineering concepts such as data management, CRUD operations, API handling, and frontend-backend integration.

## 🚀 Features

✅ Add Tasks
✅ Display All Tasks
✅ Mark Tasks as Completed
✅ Delete Tasks
✅ AI-Based Task Suggestions
✅ Modern Responsive UI
✅ JSON File Storage
✅ Flask Backend APIs

## ⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone <repository-link>

## 2️⃣ Open Project Folder
cd TO_DO_APP 

## 3️⃣ Install Flask
pip install flask

## 4️⃣ Run the Application
python app.py 

## 5️⃣ Open in Browser
http://127.0.0.1:5000 

## 🎯 Application Workflow
Users can enter a task and add it to the list.

## ✅ Mark Complete

Clicking on a task marks it as completed.

## ❌ Delete Task

Tasks can be removed using the delete button.

## 🤖 AI Suggest

The AI Suggest button automatically generates productivity-related tasks.

## 🧩 Backend APIs
### Route	            Method	        Description
/	                GET	            Home Page
/get_tasks	      GET	            Fetch all tasks
/add_task	        POST	          Add new task
/delete_task	    POST	          Delete task
/toggle_task	    POST	          Mark complete
/ai_suggest	      GET	            AI task suggestion
